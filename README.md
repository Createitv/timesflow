# TimesFlow

[中文说明](./README.zh-CN.md)

TimesFlow is an automatic desktop time tracking app for macOS and Windows. It helps users understand where their time goes, analyze app usage patterns, improve focus, and build better productivity habits without starting or stopping timers manually.

## What TimesFlow Does

TimesFlow automatically tracks the active application on your computer in the background and turns that activity into structured productivity data. Instead of asking users to run timers, label sessions manually, or maintain spreadsheets, it builds a continuous timeline of actual work behavior.

Core capabilities include:

- Automatic time tracking for desktop app usage
- Focus sessions and distraction awareness
- Productivity dashboard with top apps and category breakdowns
- Timeline view of application activity across the day
- GitHub-style heatmap for long-term activity patterns
- Goal tracking for focus time, active time, and category limits
- Productivity reports for weekly, monthly, and custom ranges
- CSV export for personal analysis and backups
- Local-first data storage with optional cloud sync for Pro users

## Who TimesFlow Is For

TimesFlow is built for people who want a screen time tracker, app usage tracker, productivity monitor, or automatic time tracker on desktop without heavy manual setup.

Typical users include:

- Solo founders who want to see how much time goes to shipping, admin, meetings, and distractions
- Remote workers who need clearer visibility into deep work versus fragmented work
- Students who want to understand study time and digital distraction patterns
- Designers and developers who switch between many tools and want a real activity timeline
- Knowledge workers comparing manual time tracking tools with automatic tracking alternatives

## Key Product Positioning

TimesFlow focuses on practical automatic tracking, readable analytics, and a simpler workflow than timer-based tools.

- Automatic instead of manual: no need to start and stop timers for normal usage
- Local-first by default: tracking data stays on the device unless the user enables sync
- Built for clarity: dashboard, timeline, heatmap, reports, and goals work together
- Cross-platform desktop support: macOS and Windows
- Suitable for personal productivity, self-analytics, and behavior improvement

## Main Features

### Automatic Time Tracking

TimesFlow samples the frontmost active app in the background, allowing users to understand real app usage without extra friction.

### Dashboard Analytics

Users can review total tracked time, top applications, category distribution, and progress toward daily goals.

### Timeline View

A chronological activity log helps users inspect how the day unfolded and when context switching happened.

### Focus Mode

Focus sessions help users block out distraction and build intentional work blocks.

### Activity Heatmap

A yearly heatmap gives a fast way to inspect consistency and long-term work patterns.

### Goals and Reports

Goals and recurring reports make the data actionable instead of decorative.

- The app can work offline
- Cloud sync is optional

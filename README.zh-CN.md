# TimesFlow

[English README](./README.md)

TimesFlow 是一款面向 macOS 和 Windows 的自动时间追踪应用，用来帮助用户了解时间花在了哪里、分析应用使用习惯、提升专注效率，并在不依赖手动计时的前提下持续改进生产力。

- 使用文档：https://www.timesflow.app/docs
- 版本发布页：https://www.timesflow.app/releases

## TimesFlow 是做什么的

TimesFlow 会在后台自动记录当前前台活跃应用，把原始活动数据整理成更容易理解的生产力信息。用户不需要像传统计时工具那样频繁开始、暂停或切换任务，应用会自动生成真实的时间线和使用分布。

核心能力包括：

- 自动追踪桌面应用使用时间
- 专注模式与干扰识别
- 生产力仪表盘与分类分析
- 按时间顺序查看每日活动时间线
- GitHub 风格年度活动热力图
- 每日目标与分类目标管理
- 周报、月报和自定义周期报告
- CSV 数据导出
- 本地优先存储，专业版支持云同步

## 产品定位

TimesFlow 的核心定位是“自动追踪 + 可读分析 + 低操作成本”。

- 相比手动计时软件，更轻量，不需要频繁操作
- 相比系统自带屏幕时间，更适合工作和生产力分析
- 相比纯云端统计产品，更强调本地优先和用户数据控制
- 相比只会出图表的工具，更重视目标、报告和长期行为改善

## 主要功能

### 自动时间追踪

自动识别当前前台应用，持续记录真实使用时长，降低用户输入成本。

### 仪表盘分析

快速查看总活跃时长、Top 应用、分类占比和目标进度。

### 时间线

按顺序展示一天内的应用切换和工作片段，方便分析上下文切换。

### 专注模式

帮助用户建立专注时段，减少干扰，形成稳定的工作节奏。

### 活动热力图

用年度热力图查看长期活跃模式和行为一致性。

### 目标与报告

让追踪数据真正用于复盘，而不是停留在“看上去很多图表”的层面。

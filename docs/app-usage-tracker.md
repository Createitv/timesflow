# App Usage Tracker

An app usage tracker helps users understand which tools consume time across the workday. TimesFlow is designed to make that data readable and useful.

## What Users Can See

- top apps
- time by app category
- timeline of app switching
- daily and weekly patterns
- reports for longer review periods

## Why Users Look for This

App usage tracking matters for people who want to:

- reduce distraction
- understand software habits
- compare creative work time versus admin time
- analyze how much time goes to meetings, browsers, code editors, and communication tools

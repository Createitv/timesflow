# 开发者时间追踪工具

开发者对时间追踪软件的需求，往往和计费场景不一样。他们更关心深度工作时间、工具切换、分心模式，以及一天到底是怎么被切碎的。

## 为什么 TimesFlow 适合开发者

- 自动记录编码相关软件使用情况
- 更容易看清上下文切换
- 帮助区分深度工作时间和碎片化时间
- 用报告和热力图观察长期模式

## 更适合哪些人

- 独立开发者
- 初创团队工程师
- Indie Hacker
- 做个人复盘的远程研发人员

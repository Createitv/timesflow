# TimesFlow Product Overview

TimesFlow is a desktop productivity and automatic time tracking app designed for people who want to understand computer usage without relying on manual timers.

## Product Summary

TimesFlow tracks active app usage on desktop, organizes it into a readable activity timeline, and helps users turn raw behavior into useful insight. It is positioned for people searching for a time tracker, productivity tracker, app usage tracker, screen time alternative, or local-first productivity tool.

## Main Use Cases

- Understand where work hours actually go
- Identify distraction patterns across apps
- Build consistent focus habits
- Review weekly and monthly productivity patterns
- Compare productive versus distracting time
- Export time data for personal review or reporting

## Feature Areas

- Automatic tracking
- Daily dashboard

- Activity timeline

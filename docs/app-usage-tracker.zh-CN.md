# 应用使用统计工具

应用使用统计工具的核心价值，是帮助用户看清一天时间到底被哪些软件消耗。TimesFlow 的设计目标就是让这些数据更容易理解，也更有行动价值。

## 用户能看到什么

- Top 应用
- 按分类统计的使用时间
- 应用切换时间线
- 每日和每周使用模式
- 更长周期的数据报告

## 为什么用户会搜索这类产品

这类工具适合想要：

- 减少分心
- 理解软件使用习惯
- 对比创作时间和事务性时间
- 分析浏览器、会议、代码编辑器和沟通工具分别花了多少时间

# Productivity Tracker

TimesFlow is positioned as a productivity tracker for desktop users who want to move from vague impressions to concrete behavioral data.

## What Makes It a Productivity Tracker

- app usage tracking
- dashboard analytics
- timeline review
- focus sessions
- goals
- reports
- heatmap-based consistency review

## Why This Matters

A productivity tracker should not only collect data. It should help users interpret how work happens and where improvement is possible. TimesFlow is designed around that practical review loop.

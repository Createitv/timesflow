# 自动时间追踪软件

搜索自动时间追踪软件的用户，通常是希望拿到有价值的数据，而不是反复手动开关计时器。TimesFlow 就是围绕这种工作流设计的。

## TimesFlow 能做什么

- 自动记录桌面当前活跃应用
- 帮助用户回看时间到底如何花掉
- 降低手动输入成本
- 同时支持 macOS 和 Windows

## 为什么用户会搜索这类产品

这类用户通常想要：

- 比手动计时软件更低摩擦
- 更真实的工作习惯可视化
- 能区分高效时间和干扰时间
- 可以长期在后台运行的桌面工具
